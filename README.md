# Phalk Profiles Plugin for Playnite

This is a plugin for the [Playnite](https://playnite.link/) game library manager. It allows you to automatically sync your game library and playtime directly with your **Phalk Profiles** account ([www.phalk.net/profiles](https://www.phalk.net/profiles)).

---

## 🚀 Features

The plugin automates the export of your game data from Playnite to the Phalk Profiles platform. With it, you can:
* **Share Your Library:** Showcase your entire game collection in one single place.
* **Sync Playtime:** Keep your game hours counter updated automatically on your web profile.
* **Sync Achievements:** It is compatible with achievements from the addon [Playnite Achievements](https://github.com/justin-delano/PlayniteAchievements/)
* **Centralization:** Consolidate statistics from multiple platforms (Steam, Epic, GOG, PlayStation, Xbox, Nintendo etc.) that you already manage via Playnite onto your **Phalk Profiles** page.

---

## 🛠️ How to Use

### 1. Prerequisites
* **Playnite** installed (Desktop mode).
* An active **Phalk Profiles** account.

### 2. Installation
1. Go to the **Releases** tab of this GitHub repository.
2. Download the latest file with the `.pext` extension.
3. With Playnite open, double-click the downloaded `.pext` file or drag and drop it into the Playnite window.
4. Confirm the plugin installation and restart Playnite when prompted.

### 3. Configuration
1. In Playnite, open the main menu (Playnite icon in the top-left corner) > **Add-ons** > **Extension Settings** > **Generic**.
2. Locate **Phalk Profiles** in the list.
3. Enter your credentials or the API key provided by the [Phalk Profiles](https://www.phalk.net/profiles) website to authenticate the plugin.
4. Save your changes.

### 4. Synchronization
* The plugin will run seamlessly and you will be able to check your games on the Phalk Profiles website right away.
* It will also auto-update the data automatically when you end a play session.
